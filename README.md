# Malaria Diagnosis using Convolutional Neural Networks

This repository contains a **single-file implementation** of a malaria diagnosis system using a **Convolutional Neural Network (CNN)** built with **TensorFlow (Keras)**.  
The model classifies microscopic cell images to detect malaria infection.

## Overview

- Implemented an end-to-end CNN-based malaria diagnosis model in a single Python file  
- Applied data preprocessing and data augmentation techniques  
- Implemented a **AlexNet-style CNN architecture**    
- Monitored training using **TensorBoard**  
- Achieved **95.17% testing accuracy** on unseen test data  
- Evaluated performance using **confusion matrix visualization**
